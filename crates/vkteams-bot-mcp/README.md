## Quick Start

### 1. Set Environment Variables

```bash
export VKTEAMS_BOT_API_TOKEN=your_token_here
export VKTEAMS_BOT_API_URL=https://your-api-url
export VKTEAMS_BOT_CHAT_ID=your_chat_id
```

### 2. Build and Run

```bash
cargo build --release
./target/release/vkteams-bot-mcp
```

### 3. Example MCP Requests

#### Send a text message

```json
{
  "tool": "send_text",
  "params": {
    "text": "Hello, world!"
  }
}
```

#### Upload a file from base64

```json
{
  "tool": "upload_file_from_base64",
  "params": {
    "file_name": "document.txt",
    "base64_content": "SGVsbG8gV29ybGQh",
    "caption": "Here's the document you requested"
  }
}
```

#### Upload text as a file

```json
{
  "tool": "upload_text_as_file",
  "params": {
    "file_name": "script.py",
    "text_content": "print('Hello from Python!')",
    "caption": "Python script generated by AI"
  }
}
```

## Usage

The server communicates via the [MCP protocol](https://github.com/modelcontextprotocol/rust-sdk) over standard input/output.
You can connect it to LLM agents, automation scripts, or use it as a standalone service.

### Supported Tools

#### Core Bot Operations

- `send_text` â€” Send a text message to the chat
- `self_get` â€” Get bot information
- `chat_info` â€” Get chat information
- `file_info` â€” Get file information
- `events_get` â€” Get events from the chat

#### Enhanced File Upload

- `upload_file_from_base64` â€” Upload file from base64 encoded content
- `upload_text_as_file` â€” Create and upload text content as a file
- `upload_json_file` â€” Create and upload structured JSON files
- `upload_csv_file` â€” Generate and upload CSV files from data
- `upload_code_file` â€” Upload source code with syntax highlighting
- `upload_multiple_files` â€” Batch upload multiple files at once
- `validate_file_before_upload` â€” Validate file content before uploading
- `upload_with_conversion` â€” Upload with automatic format conversion

#### Message Management

- `send_file` â€” Send file to chat

- `delete_message` â€” Delete message from chat
- `pin_message` â€” Pin message in chat

- `set_chat_avatar` â€” Set chat avatar

- `block_user` â€” Block user in chat

## File Upload Capabilities

The server includes advanced file handling with automatic validation, format conversion, and batch processing:

### Supported File Types

- **Text files**: `.txt`, `.md`, `.json`, `.xml`, `.csv`, `.log`
- **Code files**: `.rs`, `.py`, `.js`, `.ts`, `.html`, `.css`, `.sql`
- **Images**: `.png`, `.jpg`, `.jpeg`, `.gif`, `.svg`, `.webp`
- **Documents**: `.pdf`, `.doc`, `.docx`, `.xls`, `.xlsx`
- **Archives**: `.zip`, `.tar`, `.gz`, `.7z`
- **Audio/Video**: `.mp3`, `.wav`, `.ogg`, `.mp4`, `.avi`

### File Validation

- Automatic filename sanitization
- File size validation (max 10MB)
- Content type detection
- Base64 encoding/decoding
- MIME type identification

### Advanced Features

- Batch file uploads (up to 10 files)
- Automatic format conversion (text â†” JSON â†” CSV)
- Code syntax highlighting
- File metadata extraction
- Progress reporting for batch operations
